# Automated Fertilizer Sprayer System

## 📌 Overview
The Automated Fertilizer Sprayer is designed to efficiently distribute fertilizers to plants based on soil conditions. This system reduces manual labor and ensures proper nutrient supply for improved crop growth.

## 🎯 Objective
To develop an automated system that sprays fertilizer only when required, optimizing usage and improving agricultural productivity.

## ⚙️ Working Principle
The system monitors soil conditions using sensors (such as soil moisture or nutrient indicators). When the required condition is met (e.g., low nutrient or dry soil), the system activates a pump or motor to spray fertilizer.

The spraying mechanism is controlled using a microcontroller or electronic switching circuit.

## 🛠️ Components Used
- Soil Moisture Sensor  
- Pump / Sprayer Motor  
- Microcontroller (Arduino / Raspberry Pi Pico) OR Transistor/Relay Circuit  
- Power Supply  
- Pipes / Nozzle System  

## 🚀 Features
- Automated fertilizer distribution  
- Reduces manual effort  
- Efficient resource utilization  
- Improves crop yield  
- Low-cost implementation  

## 🔧 Technologies Used
- Embedded Systems / Basic Electronics  
- Sensor Integration  
- Control Systems
